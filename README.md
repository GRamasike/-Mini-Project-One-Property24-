# 📌 Property 24 SQL Mini Project – Task Overview Data Engineering

# 🏘️ Project Background
This project focuses on exploring and analysing a large residential property dataset using SQL. The dataset simulates a real estate platform similar to Property21, containing approximately 100 000 property records across all nine provinces of South Africa.<br>

The goal of this project is to practice data exploration and querying using SQL without modifying the dataset.

# 📁 Dataset
- Property prices 
- Location (Country, Province, City)
- Bedrooms and bathrooms
- Other property features

  The dataset is used to sumulate a real world estate data environmnent.

# 🎯 Project Objective
Use # SQL SELECT queries 

# 📜Rules
• Do not change the table structure. <br>
• Do not delete or update any records. <br>
• Only use read-only queries (SELECT).

# What Is Being Assessed
• Correct use of SQL syntax.<br>
• Proper filtering and sorting of data. <br>
• Ability to interpret business-style questions into SQL queries.

# Deliverables
Submit a document or SQL file containing all your queries, clearly numbered to match the question sheet.

# Estimated Difficulty
Beginner to Intermediate SQL level.
